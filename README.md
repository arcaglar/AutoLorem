Auto-Lorem Dummy Text/Image Generator - Native JS
========================================================

How To Use
----------

Download and add

    <script src="path/to/lorem.js"></script>

or
    <pre>
        npm i auto-lorem
    </pre>

If you want to lorem text generate:

    <div auto-lorem="12"></div>

Dummy Images
------------

To use Lorempixel:

Usage:

    <img src="" auto-lorem="1920-1080">
    <img src="" auto-lorem="400-200">
